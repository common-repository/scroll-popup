Responsive Scroll PoPup Ads Manager for WordPress.

== Description ==

Scroll Popup allows you to slide popups from all four corners of your website. You can choose in which corner you would like the popup to appear and also the direction you would like the popup to slide from.


* [Live demo!&raquo;](shttp://themepoints.com)

== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see "Scroll PoPup" menu at left side on WordPress dashboard and find "Scroll PoPup Ads Settings"<br />
<br />

<strong>Using FTP</strong>
1.Extract tp-scroll-popup-ads.zip to your computer.
2.Upload the tp-scroll-popup-ads directory to your wp-content/plugins directory.
3.After activate plugin you will see "Scroll PoPup" menu at left side on WordPress dashboard and find "Scroll PoPup Ads Settings"
